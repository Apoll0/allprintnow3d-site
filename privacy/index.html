Privacy Policy — AllPrintNow3D

Effective Date: 17 February 2026   •   Last Updated: 17 February 2026

Pinterest API / Developer Review (Ultra-safe)

1. Overview

AllPrintNow3D (“we”, “our”, or “us”) operates a private internal tool used solely to manage and publish content to our own Pinterest business account through the Pinterest API.

This Privacy Policy explains how information is processed in connection with this integration. This application is not available to the public and is not used by external users.

2. Purpose of the Application

The application is developed exclusively for internal business use to:

  • Authenticate our own Pinterest business account
  • Create and publish Pins
  • Manage Boards
  • Monitor publishing status

The tool does not provide services to third parties.

3. Information Processed

Our application processes only the minimum data necessary to operate our own Pinterest account, including:

  • Pinterest account authentication data (OAuth tokens)
  • Board identifiers and metadata
  • Pin identifiers and metadata
  • Content created and owned by AllPrintNow3D (images, titles, descriptions)
  • API responses required to confirm publishing status

We do not collect personal data from Pinterest users unrelated to our account. We do not request access to private user data beyond what is required to publish content to our own account.

4. No Sale or Sharing of Data

We do not:

  • Sell personal information
  • Share Pinterest data with third parties
  • Use Pinterest data for advertising networks
  • Aggregate or resell Pinterest data
  • Use Pinterest data outside the permissions granted by the Pinterest API

All data is used solely for internal content management.

5. Data Storage and Security

Access to the application is restricted to authorized personnel only. OAuth credentials and related data are stored securely and are not shared externally.

We use HTTPS and standard security practices to protect processed information.

6. Data Retention

Data is retained only as long as necessary to maintain API functionality and publishing workflows. Expired or revoked authentication tokens are deleted.

7. Compliance with Pinterest Policies

We comply with:

  • Pinterest Developer Guidelines
  • Pinterest API Terms
  • Applicable data protection laws

Pinterest data is used strictly within the scope permitted by Pinterest.

8. User Rights

Although our application does not collect personal data from external users, individuals may contact us if they believe their information has been processed in error.
